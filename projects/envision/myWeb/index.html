个人网站

一个具备内容展示、工具集成、权限控制、个性化体验的自建型全栈个人网站，面向所有人开放，目前处于项目需求分析和技术选型阶段。

需求分析

业务需求

1. 项目背景与目标

本项目旨在构建一个个人综合网站平台，用于展示内容创作（博客、项目、生活随笔）、管理工具模块（如待办、清单、歌单等），并支持个性化信息管理与对外展示。

网站需支持游客访问、个性化授权阅读、站长内容维护与扩展功能管理，可持续迭代并具备良好的部署与安全能力。

2. 目标用户

用户角色 特征 权限
站长（管理员） 唯一登录用户 全站内容管理权限，创建用户
授权用户 无注册，仅由站长创建 可访问部分保护内容
游客 所有人 可浏览公开内容、留言、评论等基础功能

3. 主要使用场景

  • 作为个人博客与作品集门户

  • 作为工具中心 / 自用网页工作台

  • 作为内容创作与记录平台

  • 支持移动端阅读与维护、跨设备同步使用

功能需求

1. 博客系统

功能 描述
博客发布 使用 Markdown 编辑器撰写，支持上传图片、自动生成 HTML
博客分类 / 标签 支持多个分类、标签管理与搜索
归档展示 按月自动归档，升序排列
评论功能 游客可评论，输入昵称+邮箱+站点（可选），头像使用 Gravatar
权限控制 支持“公共 / 保护”两类权限，保护博客需授权用户登录访问
浏览量统计 每篇博客自动计数，生成预计查看时间，后台可查看热度
博客置顶 支持设置置顶博客优先展示

2. 项目展示模块

功能 描述
项目新增 / 编辑 / 删除 管理员可操作，结构与博客类似
项目属性管理 包含名称、技术栈、封面图、简要描述、仓库地址
状态标识 支持设想 / 进行中 / 已凉状态标记
权限控制 游客可查看，无评论，无互动入口

3. 工具模块

TodoList
功能 描述
创建 / 修改 / 删除事项 基础功能
置顶桌面支持 计划通过 PWA/Electron 实现独立插件
今日高亮 可用于首页卡片形式展示
歌单模块
功能 描述
支持播放 / 暂停 / 切歌 本地优先，从 /music 文件夹读取
网络搜索播放 若本地无资源，支持在线播放
扩展能力 支持读取歌词 / 歌词高亮 / 播放列表（预留接口）
书单 / 影单
功能 描述
自定义记录 结构与博客一致，支持分类、标签、评分
阅读 / 观看状态 支持“想读 / 读过”、“想看 / 看过”标记（可扩展）
简评 + Markdown 详情页 可填写内容与阅读笔记 / 观后感

4. 留言板模块

功能 描述
游客留言 填写昵称、内容、邮箱和站点
管理员回复 通过后台操作，支持一对一回复
邮箱通知 若游客填写邮箱，则在回复后触发通知邮件

5. 关于我页面

功能 描述
静态展示 展示站长介绍、经历、联系方式、建站初心等
管理员可编辑 使用 Markdown 编辑，可首次创建或随时更新

6. 网站导航与首页模块

功能 描述
网站导航 类似浏览器导航栏，支持最多 20 个链接收藏展示，仅管理员可见
访问统计 网站右下角显示总访问人次
首页内容分区 左侧简介/搜索/标签区，右侧模块卡片（Todo、导航等）
友情链接 模块预留，未来可启用链接管理功能

7. 权限与用户系统

功能 描述
管理员登录 通过 JWT 登录，拥有所有管理权限
用户创建 用户仅由站长创建，无开放注册入口
游客访问 默认权限为只读，访问公共页面与留言/评论接口

8. 安全与部署

安全需求 措施
接口限速 对评论/留言等敏感接口进行频率限制
CDN 加速 接入 Cloudflare 实现静态资源缓存、访问优化、防攻击
防盗链 图片等资源设置防盗链规则
静态资源压缩 配置前端打包优化，减小带宽成本
日志审计 后端记录所有重要操作日志，便于排查问题

技术选型

系统架构

层级 技术栈 / 工具 说明
前端 Vue 3 + Vite + Vue Router + Pinia 组件化结构、响应式、自定义主题
后端 Spring Boot + MyBatis Plus + JWT RESTful 接口、权限控制、业务逻辑实现
数据库 MySQL + Redis（可选） 数据持久化 + 缓存优化
编辑器 Markdown 编辑器（如 Toast UI） 博客 / 项目 / 关于页内容编辑
部署 Nginx + Cloudflare + 域名解析 CDN加速、防攻击、HTTPS、静态代理

整体架构

前端（Vue 3） <--> Nginx反向代理 <--> 后端（Spring Boot）<--> MySQL数据库

技术选型

前端技术栈

  • Vue 3 - 渐进式JavaScript框架
  • Element Plus UI - 现代化的UI组件库
  • Vite - 下一代前端构建工具
  • Vue Router - 官方路由管理器
  • Pinia - Vue 3的状态管理库
  • Axios - HTTP客户端
  • ECharts - 数据可视化图表库
  • PDF-lib - PDF生成工具

后端技术栈

  • Spring Boot 3 - 应用开发框架
  • Spring Security - 安全框架
  • Spring Data JPA - 数据访问层
  • MySQL 8 - 关系型数据库
  • JWT - 用户认证
  • WebSocket - 实时通信
